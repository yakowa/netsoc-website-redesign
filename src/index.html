Services

Free Cloud Hosting

Netsoc provides free Cloud hosting services available to all UCC students. If you'd like to avail of these services, feel free to try them out via Netsoc Cloud. You can request free VPS's (Virtual Private Servers) or Containers. These allow you to run whatever software stack you want on our servers.

Free netsoc.cloud Domains

When you sign up to our cloud services via Netsoc Cloud you'll be able to claim *.netsoc.cloud domains that point to your cloud instances.

If you'd rather a custom domain, you can register one and follow the steps on Netsoc Cloud to get it to point to your instance.

Free Blog Hosting

We provide container templates that allows you to get up and running with a WordPress or GhostCMS instantly.

Free Game Server Hosting

Since we provide free containers, you can deploy any game server you want on our services by simply using the terminal provided by Netsoc Cloud!
